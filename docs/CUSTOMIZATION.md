# Quick Order Matrix - Theme Editor Customization Guide

This document outlines all the settings available in the Shopify Theme Editor for customizing the Quick Order Matrix section.

---

## General Settings

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| **Section Title** | Text | "Quick Order" | The heading displayed above the order matrix |
| **Collection** | Collection Picker | (blank) | Select which collection to display products from. Leave blank to use page context. |
| **Products per page** | Range (10-50) | 50 | Number of products to show per page with pagination |

---

## Section Layout

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| **Full width** | Checkbox | false | Make the section span the full width of the page |
| **Padding top** | Range (0-100px) | 36px | Top padding for the section |
| **Padding bottom** | Range (0-100px) | 36px | Bottom padding for the section |

---

## Display Options

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| **Show product images** | Checkbox | true | Display product thumbnail images in the table |
| **Show SKU column** | Checkbox | true | Show the SKU column in the table |
| **Variant display layout** | Select | "Compact" | Choose between:<br>• **Compact** - variants grouped per row<br>• **Expanded** - one row per variant with dedicated SKU |
| **Show stock indicators** | Checkbox | true | Display stock status colors/badges on quantity inputs |
| **Low stock threshold** | Range (1-50) | 10 | Inventory quantity at which stock is considered "low" and shows warning color |
| **Enable search/filter** | Checkbox | true | Show the search box for filtering products by name or SKU |
